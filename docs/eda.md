# 주요 인사이트
1. URL 길이가 중요한 피처가 될 수 있다.
악성 URL의 평균 길이가 정상 URL의 2배 이상
URL 길이가 악성 여부를 판단하는 중요한 단서

2. 데이터 스케일링 필요
URL 길이가 2자부터 8,000자 이상까지 매우 넓은 범위를 가지므로, 이 길이를 모델에 피처로 사용할 때는 **스케일링(Scaling)**이 필요하다고 판단됨.
